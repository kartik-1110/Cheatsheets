===> 2 types of containers
1: Value type/ primitive
2: Reference type

1: Value type / primitive
- These are copied by value
- When we assign value to variable it stores the value
- primitives are copied by their value
- Can copy values
    // let x =10 
       let y = x
       x= 20;
- Eg: Number, string, Boolean, symbol, undefined, null       

2: Reference type
- These are copied by reference or address
- When we assign value to object it store the address where the value is.
- Object are copied by their reference.
    // let x ={value: 10}
       let y = x
       x.value = 10;
- Eg: Object, function, array
